# Legacy Outputs Archive

Archived on 2026-04-24.

These files are historical outputs from the old MBPP/HumanEval and legacy GRPO runs. They are kept for reference only and are not part of the current OJ-like CodeContests/LiveCodeBench e2e path.

Current runs should write fresh outputs under `outputs/` and regenerate `data/verl/*.parquet` with `python3 -m src.data.verl_dataset`.
